# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.2] - 2024-01-15

### Added
- Initial public release
- SSH connection management
- Remote command execution capability
- Enable/disable access on remote devices
- Connection established event trigger
- Connection failed event trigger
- Command executed event trigger
- Support for password and key-based authentication
- Multiple device support
- Connection timeout handling
- Network and configuration permissions
- Secure credential storage

### Changed
- Improved error handling for connection failures
- Enhanced command validation

### Fixed
- Connection stability issues
- Command timeout edge cases

## [0.0.1] - 2024-01-01

### Added
- Initial development version
- Basic plugin structure
- Core integration with Allow2Automate
- SSH client integration
